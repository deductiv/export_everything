Field or string to be assigned to the source field on the exported event.
Syntax: [source_value|$source_field$]
Default: $source$, or if not defined, it is omitted.
Example: $name$ (saved search name)
Field or string to be assigned to the sourcetype field on the exported event.
Syntax: [sourcetype_value|$sourcetype_field$]
Default: $sourcetype$, or if not defined, it is omitted.
Example: $result.sourcetype$
The remote index in which to store the exported event.
Syntax: [index_value|$index_field$]
Default: $index$, or if not defined, the remote endpoint's default.
Field or string to be assigned to the host field on the exported event.
Syntax: [host_value|$host_field$]
Default: $host$, or if not defined, the hostname of the sending host (from inputs.conf)
