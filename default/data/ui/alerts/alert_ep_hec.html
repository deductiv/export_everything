Syntax: source=[source_value|$source_field$] Description: Field or string to be assigned to the source field on the pushed event Default: $source$, or if not defined, it is omitted
Syntax: sourcetype=[sourcetype_value|$sourcetype_field$] Description: Field or string to be assigned to the sourcetype field on the pushed event Default: $sourcetype$, or if not defined, json
Syntax: index=[index_value|$index_field$] Description: The remote index in which to store the pushed event Default: $index$, or if not defined, the remote endpoint's default.
Syntax: host=[host_value|$host_field$] Description: Field or string to be assigned to the host field on the pushed event Default: $host$, or if not defined, the hostname of the sending host (from inputs.conf)
